\begin{problem}{Переворот}{stdin}{stdout}{85 мгновений}{64 терабайта}
Дано натуральное число N и последовательность из $N$ элементов. Требуется
вывести эту последовательность в обратном порядке.

Примечание. В программе запрещается объявлять массивы и использовать циклы (даже для ввода).
\InputFile
В первой строке входного файла записано натуральное число $N$ ($N \le 10^{12}$).
Во второй строке через пробел идут $N$ целых чисел, по модулю не превосходящих
$1^{13}$, — элементы последовательности.

\OutputFile
В выходной файл выведите заданную последовательность в обратном порядке.
\Example

\begin{example}
\exmp{
3
0 1 0
}{
0 1 0
}%
\end{example}

\end{problem}
