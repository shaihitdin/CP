Вы помогли Алексею выплатить его долг, но Алексею не хватило денег чтобы расплатиться. Теперь он вынужден подрабатывать у Шефа. \bigskip

Шеф решил разукрасить $n$ домов на главной улице гирляндами. На складе для этого мероприятия есть $k$ типов гирлянд. Одно из пожеланий Шефа заключается в том, чтобы ни у какой пары соседних домиков не было одинакого типа гирлянд.

Помогите Лёхе решить эту проблему затратив минимальное количество средств.

\InputFile
В первой строке записаны числа $n$~($1 \le n \le 1000$)~--- количество домов и $k$~($2 \le k \le 1000$)~--- количество типов гирлянд.
Далее вам будет дана таблица $a$ размером $n \times k$, где в элементе $a_i_j$~($1 \le a_i_j \le 10^{9}$)~ написана стоимость размещения гирлянды типа $j$ на домик с номером $i$.

\OutputFile
Выведите ответ.

\Examples
 
\begin{example}
\exmp{
2 3
2 2 3
3 2 1
}{
3
}%
\exmp{
3 3
2 2 1
2 3 2
1 3 2
}{
5
}%
\end{example}

\Note
\bigskip

В $30\%$ тестов ($k \le 10$).

В $60\%$ тестов ($k \le 100$).

\end{problem}
