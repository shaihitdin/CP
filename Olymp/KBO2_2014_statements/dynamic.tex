Вы помогли Алексею выплатить его долг, но Алексею не хватило денег чтобы расплатиться. Теперь он вынужден подрабатывать у Шефа. \bigskip

Шеф решил разукрасить $n$ домов на главной улице гирляндами. На складе для этого мероприятия есть 3 типа гирлянд. Одно из пожеланий Шефа заключается в том, чтобы ни у какой пары соседних домиков не было одинакого типа гирлянд. Соседними домиками являются домики номера которых различаются не более чем на один.

Помогите Лёхе решить эту проблему затратив минимальное количество средств.

\InputFile
В первой строке записаны числа $n$~($1 \le n \le 1000$)~--- количество домов.
Далее вам будет дана таблица $a$ размером $n \times 3$, где в элементе $a_i_j$~($1 \le a_i_j \le 10^{9}$)~ написана стоимость размещения гирлянды типа $j$ на домик с номером $i$.

\OutputFile
Выведите ответ.

\Examples
 
\begin{example}
\exmp{
2
2 2 3
3 2 1
}{
3
}%
\exmp{
3
2 2 1
2 3 2
1 3 2
}{
5
}%
\end{example}


\end{problem}
